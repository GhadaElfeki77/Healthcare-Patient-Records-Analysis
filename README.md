# Healthcare-Patient-Records-Analysis

Healthcare analytics dashboard using SQL Server and Power BI

Project Overview

Developed an interactive healthcare analytics dashboard using SQL Server and Power BI to analyze patient encounters, insurance coverage, claim costs, and clinical trends. The project focused on transforming healthcare data into meaningful insights to support data-driven decision-making.

Objectives
Analyze patient encounter patterns.
Evaluate healthcare claim costs and insurance coverage.
Identify the most common medical conditions and encounter types.
Monitor patient demographics and healthcare utilization.
Create an interactive dashboard for business and clinical insights.
